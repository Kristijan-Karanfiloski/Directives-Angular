Directives are use to change the behavior and appearance of DOM element.

Directives can implement all lifecycle hooks.

Directives can not have template

Binding Syntax

  • Interpolation
  • Property Binding
  • Event binding

Built-in Directives

  • *ngIf: This directive allows you to conditionally show or hide elements based on a condition. For example, you could use *ngIf to show a login form only if the user is not already logged in.
  • *ngFor: This directive is used for looping over a collection of data and generating HTML elements for each item in the collection. For example, you could use *ngFor to display a list of items from an array.
  • *ngSwitch: This directive is used for conditionally rendering HTML elements based on a variable's value. It works similarly to a switch statement in other programming languages
  • *ngClass: This directive allows you to dynamically add or remove CSS classes based on a condition. For example, you could use *ngClass to apply a "highlight" class to an element when it is clicked
